Tentang Proyek

LoginApp adalah sebuah sistem otentikasi pengguna berbasis web yang dibangun menggunakan HTML, TailwindCSS v4, PHP, dan MySQL. Proyek ini mencakup halaman login, pendaftaran pengguna baru, validasi akun, serta manajemen user berbasis peran (admin dan user biasa). Tujuan utamanya adalah menyediakan mekanisme login yang aman, sistem pendaftaran terintegrasi dengan database, dan akses yang disesuaikan berdasarkan level user.

Fitur Utama

  • Form Login Dinamis + Feedback
    • Validasi email & password dari database MySQL via proses/login.php.
    • Notifikasi (toast) jika login berhasil/gagal.
    • Tombol "Lupa Password" menuju halaman kontak admin via WhatsApp.
    • Akses dibagi dua:
      • admin@gmail.com ke /dashboard/
      • Email lainnya ke /landing-page/
    • Note: untuk halaman /landing-page/ hanya contoh menampilkan hero sectionnya saja.
  • Form Register dengan Validasi & Konfirmasi Password
    • Validasi kecocokan password dan konfirmasi password di sisi client.
    • Cek apakah email sudah terdaftar (via proses/cek_email.php).
    • Data disimpan ke MySQL melalui proses/register.php.
    • Notifikasi toast saat berhasil mendaftar atau jika gagal.
  • Dashboard Admin (Role-Based Access)
    • Akses khusus untuk admin.
    • Menampilkan data user.
  • Struktur Folder Modular & Rapi
    • Folder proses/ untuk semua logic backend (login, register, logout, dll).
    • Halaman landing-page/ untuk pengguna biasa.
    • Folder img/ untuk aset visual.
  • Fitur Tambahan
    • Validasi frontend menggunakan JavaScript.
    • Feather Icons untuk tampilan modern.
    • Toast notifikasi dengan animasi.
  • Cocok digunakan untuk:
    • Sistem login admin dan pengguna biasa.
    • Website internal perusahaan.
    • Sistem registrasi event/sekolah/komunitas
    • Template dasar untuk proyek berbasis user management
